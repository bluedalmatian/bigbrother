MAJOR CHANGES FOR EACH VERSION
======================================================================================================================================================================
2.1	Fixed rendering bug affecting video size and Z index ordering when using minimalUI mode, fixed issue where recording could be corrupted when BB restarted
	added networking dependency to System D unit file to prevent BB starting before network up, added feature if video playback in mirroring page freezes page reloads,
	fixed permissions on mirroring video segment files so not world readable, added event snapshot images to mirror webpage event log viewer, updated AI model
	restored Firefox compatability

2.0 	Added AI event monitoring for detecting People & Vehicles, added minimal UI for mirroring

1.3 	Fixed issue with ffmpeg5+ compatability where ffmpeg could hang if camera connection is lost. BigBrother is no longer compatible with ffmpeg 4 and earlier

1.2	Added ability to specify HLS bit rate. Added support for MKC, WEBP & MXG containers. Bug fixes: compatability with PHP8, ffmpeg5

1.1	Updated mirror UI, FreeBSD compatability, bug fixes for Python 3 compatability, checks if ffmpegcommand option is valid executable

1.0	Adopted video.js for HLS cross browser support improved HLS performance
